Freedom Project

The “Freedom Project” for SEP10 is all about making a webpage for something you are interested in and informing people on how technology in different fields can help benefit society. The webpage is composed of an existing software and hardware section, context and a future section about what you want to make in the future. The purpose of this blog is to keep track of what I was doing for the freedom project and show the audience my thinking process throughout this project.

For my project, I am making a website about construction using HTML, CSS, Github and my tool which is Materialize. I researched a lot on Construction and the technology that is applied to this topic. Then I used the informations I gathered on Construction to make a website that presents my researches.

Some challenges I encountered while working on the freedom project is finding the correct color and images. Some big problems I encountered was not getting the cards to have a equal height. This problem made my website look a bit weird since the cards aren't all the same height. However, I solved this problem by asking Mr.Mueller. He told me to use this css class height: 100%; this css class basically makes all your card element the same height. Some takeaways I had was to look up templates when you don't know how to start your website because the template actually helped me out by a ton. I also learned that you should always ask for help when you need it or you would take forever to solve a problem by yourself. In the future, I think I would try to make a better website all by myself instead of relying on templates.
